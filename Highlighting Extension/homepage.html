Tag Assistant

Type one command per line. Names are comma-separated. Lines starting with # are comments. Nothing is saved until you confirm.

Rules & examples

Create one or many tags:

create alpha, beta, gamma

Create a parent and attach many new or existing children (existing tags get that parent added):

create parent tag-p with children tag-1, tag-2, tag-3, tag-4, tag-5

Delete by name list:

delete old-tag, other-tag

Delete all tags whose names start with a prefix (case-insensitive):

delete starting with th

Relations (use exact tag names):

set parents of child-tag to parent-a, parent-b
set children of parent-tag to kid-1, kid-2
add parent scholarship to darwell-stone
remove parent foo from bar
add children c1, c2, c3 to parent-p
remove children x, y from parent-p

Rename & color:

rename old-name to new-name
set color my-tag #4CAF50
