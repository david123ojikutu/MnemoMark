# Firebase Project Name Suggestion

## Recommended Project Name

**`hierarchical-highlighter`** or **`hierarchical-highlighter-app`**

## Why This Name?

- **Consistent**: Matches the application name "MnemoMark"
- **Descriptive**: Clearly indicates what the project is for
- **URL-friendly**: Uses lowercase with hyphens (Firebase project IDs must be lowercase and can contain hyphens)
- **Professional**: Simple and clear

## Alternative Names

If the above is taken, you could use:
- `hierarchical-highlighter-dev` (for development)
- `hierarchical-highlighter-prod` (for production)
- `highlighting-app` (shorter version)
- `hierarchical-tag-highlighter` (more descriptive)

## Note

The Firebase project ID will be used in:
- Your Firebase config (`projectId` field)
- The auth domain (`your-project-id.firebaseapp.com`)
- The storage bucket (`your-project-id.appspot.com`)

So choose something you'll remember and that represents your application well!
